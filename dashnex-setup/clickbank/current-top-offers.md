## CLICKBANK ACCOUNT BASICS

### Setting Up:
1. Go to clickbank.com → Sign Up
2. Create account nickname (your brand)
3. Get APPROVED as affiliate for offers
4. Generate unique hop links with tracking IDs

### Finding Offers:
- Browse marketplace by category
- Check "Gravity" score (higher = more affiliates making sales)
- Look for approved affiliates status
- Check commission % and average sale price

### Key Links:
- Account: https://accounts.clickbank.com
- Marketplace: https://www.clickbank.com/marketplace
- API Docs: https://support.clickbank.com/en/articles/10535400-clickbank-apis
